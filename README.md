## Instalasi

1. Pastikan Anda memiliki Python diinstal. [Download Python](https://www.python.org/downloads/)

2. Clone repositori ini:

    ```bash
    git clone https://github.com/Irfiyandaabidin/crud-django.git
    cd nama-proyek
    ```
